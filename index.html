WebSystems@Brown

Our group at Brown University is trying to develop tools to understand how and why Web Quality of Experience Degrades when the client is a low end device, and to build on that information to develop systems that automatically generates webpages that are better tailored for low-end devices.

Webmedic

Developing a system that intelligently trims Javascripts from a webpage such that it maximizes Quality of Experience while minimizing its Memory consumption

Android Web Memory

Developing and discovering techniques and tools to understand and compare memory utilization of webpages.

Improving Website Performance Benchmarking

Developing tools that deterministically and realistically records and replays the webpages

Web Javascript Annotations

Currently trying to develop tools that automatically annotates a given javascript with its potential function and its footprint
